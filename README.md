Описание проекта:
    Данный проект представляет собой текстовую игру "Лабиринт сокровищ", в которой игрок с помощью консольных команд блуждает по комнатам, собирает артефакты в инвентарь, разгадывает ребусы. Главной целью игры является открытие комнаты с сокровищем.
    
Инструкция:
    Для запуска игры удостоверьтесь в том, что правильно выполнили следующие пункты:
        1. Клонируйте репозиторий с игрой (git clone https://github.com/SNWZ09/project1_Zaremba_Stepan_M25-555)
        2. Перейдите в директорию проекта (cd project1_Zaremba_Stepan_M25-555)
        3. Установите зависимости (make install или poetry install)
        4. Запустите проект (make project или poetry run project)
